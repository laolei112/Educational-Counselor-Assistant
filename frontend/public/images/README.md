# 二维码图片说明

请将以下二维码图片放置在此目录下：

1. **wechat-qr.png** - 微信二维码图片
2. **whatsapp-qr.png** - WhatsApp二维码图片

## 图片要求

- 格式：PNG 或 JPG
- 建议尺寸：至少 200x200 像素
- 文件大小：建议小于 500KB

## 如何生成二维码

1. **微信二维码**：
   - 打开微信，进入"我" -> "设置" -> "账号与安全" -> "二维码名片"
   - 保存二维码图片

2. **WhatsApp二维码**：
   - 打开WhatsApp，进入"设置" -> "二维码"
   - 保存二维码图片

将保存的图片重命名为对应的文件名并放置在此目录下即可。
